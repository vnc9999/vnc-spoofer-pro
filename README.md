## 🔑 Hướng dẫn sử dụng chi tiết

### Bước 1: Kích hoạt bản quyền
1. Chạy file `VNC_Spoofer_Pro.exe` với quyền **Administrator**.
2. Nhập mã License Key (Định dạng: `VNCN-LKSP-XXXX-XXXX-XXXX`).
3. Nhấn **KÍCH HOẠT**. Hệ thống sẽ tự động liên kết với HWID gốc của máy bạn.

### Bước 2: Thực hiện Spoofing
1. Chọn Tab **SPOOFING**.
2. Tích chọn các mục cần thay đổi (Khuyên dùng: Chọn tất cả trừ MAC nếu chơi ở quán Nét).
3. Nhấn **BẮT ĐẦU ĐỔI ĐỊNH DANH**. 
4. Chờ cho đến khi Log hiển thị `[OK] Hoàn tất`.

### Bước 3: Dọn dẹp dấu vết (Quan trọng)
1. Chuyển sang Tab **DỌN DẸP**.
2. Chọn **System Traces** để xóa sạch lịch sử hoạt động của hệ thống.
3. Chuyển sang Tab **MẠNG** để dọn dẹp bộ nhớ đệm DNS.

## 🛠️ Phần mềm & Công cụ tích hợp
Dự án sử dụng các công cụ chuyên dụng được nhúng sẵn để đảm bảo hiệu quả cao nhất:
*   **AMIDEWINx64:** Công cụ can thiệp sâu vào tầng DMI/BIOS.
*   **VolumeID64:** Công cụ từ Microsoft Sysinternals để thay đổi Disk Identifier.
*   **MAS (Microsoft Activation Scripts):** Tích hợp sẵn để hỗ trợ kích hoạt Windows sau khi Spoof.

## ⚠️ Lưu ý về UEFI/BIOS
Để việc Spoof đạt hiệu quả cao nhất, hãy đảm bảo:
*   Nếu dùng **UEFI**: Định dạng ổ cứng nên là **GPT**.
*   Nếu dùng **BIOS**: Định dạng ổ cứng nên là **MBR**.
